# Portfolio
Portfolio Website - Adil Gilani
This is a personal portfolio website of Adil Gilani, a Frontend Developer specializing in Angular, TypeScript, and modern UI frameworks like Tailwind CSS and Bootstrap. The website showcases skills, services, projects, and provides a contact form for potential clients or collaborators.
